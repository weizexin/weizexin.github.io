GMOJ P3504 【运算符】

2019-12-14 14:08:36

Description

定义新运算 “$!$” ，运算规则如下：

$a!b = a!(b-1) \times (a-1)!b ( a > 0,b > 0 )$
$a!b = 1 ( a = 0 )$
$a!b = a ( b = 0 )$

。

给你两个正整数 $n$ 和 $k$ ，现在请你求出 $n!k$ 的因子个数 $\mod (10^9+9)$ 。

$1\leq n\leq 1000,1 \leq k \leq 100$ 。
